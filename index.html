OctoDeck

Create a new deck View demo GitHub

Presentations stored in URLs. No files, no servers, no accounts. Just markdown in a hash.

How it works

Write slides in markdown. The entire deck gets compressed and encoded into the URL hash. Share the URL â€” share the presentation.

Features

  • Markdown syntax for slides
  • Live editing with instant preview
  • Customizable styles
  • Undo/redo via native browser history
  • No backend, no database, no tracking
  • Open source (MIT)

Usage

Create a new deck or view demo.

View mode controls

  • Enter - start presentation
  • Esc - exit presentation
  • Arrow keys - navigate slides
  • Space - next slide

Tech

Built with Preact and Bun. Source code on GitHub.

Analytics by GoatCounter.

  • Works only in modern browsers
