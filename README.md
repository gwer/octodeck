# OctoDeck

A tiny (~15 KB) presentation engine. Offline-ready. Slides stored in URLs. No files, no servers, no accounts. Just markdown in a hash.

https://gwer.github.io/octodeck/

Powered by Preact and Bun.

## Features

- Markdown syntax for slides
- Live editing with instant preview
- Customizable styles
- Undo/redo via native browser history
- Offline-first: once loaded, you can reopen and keep using it without a connection
- No backend, no database, no tracking

## Usage

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```
